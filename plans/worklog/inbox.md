# 工作记录缓冲区

用途：

- 作为原始工作记录入口
- 允许记录过程性、探索性、未压缩的信息
- 作为压缩进 `plans/PROJECT_MEMORY.md` 和 `plans/NEXT_ACTIONS.md` 的输入缓冲层

使用规则：

- 先记录，再压缩
- 不把本文件当长期记忆
- 达到压缩阈值后，必须清理、分流、覆盖压缩
- 只保留少量尚未定型但仍有价值的内容

建议记录格式：

```text
## 条目
- 主题：
- 类型：idea / decision / risk / task / observation / discard
- 内容：
- 是否进入长期记忆：yes / no / maybe
- 是否进入当前行动：yes / no / maybe
```
